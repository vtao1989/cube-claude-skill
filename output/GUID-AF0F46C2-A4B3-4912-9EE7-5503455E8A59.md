# Scenario Editor

Editing a scenario is possible through Scenario Editor.
Editing a scenario means defining a value for each key in the project and of a
current application, in preparation for running that scenario. Scenario Editor
is opened when double-clicking on a particular scenario in Scenario Manager.

![](GUID-D9B2FB35-414D-42B8-AEF5-0A21B963D5CA-low.jpg)

The example above demonstrates some of the different control
types available for keys. The full set of controls available are spin, edit,
combo list, and radio button list. See
[Keys](GUID-99492C90-82FF-48E2-AB5E-B3D9F83B5A27.html) and
[Working with Scenario Keys](GUID-55072C8F-26B4-49C5-A6F4-F85726B3D64B.html) for more information on setting
up keys.

In CUBE 7, you can view/edit keys from a given scenario
without the need of double-clicking on that scenario. To switch viewing/editing
keys for another scenario, use the crumbs/drop-down next to Base scenario and
select the appropriate scenario. E.g., below if you are under Base scenario,
you can easily go to a child > child scenario which is Y2025 > PlanA or
PlanB directly from the current scenario and change the settings.

![](GUID-0C98F7CE-FF59-400B-907B-6AE06A0F58C4-low.jpg)

Scenario Editor has a set of options available in its
toolbar. The first two options are to 'Save' scenario settings and 'Refresh'
scenario keys/settings. The - and + signs are to collapse/expand the list of
keys which are grouped by different categories. E.g, we have four categories as
per above image which are: System, Network, Demographic and Generation.

![](GUID-84B83AEB-F5B9-49FA-99E5-2DF0842198C5-low.jpg)

There is also a Run button on the dialog. This will run the
currently selected application using the key values as entered on the dialog.
For the run to be allowed, you must specify a value for each key used by the
selected application.

![](GUID-2A62AAF1-929C-4825-B0EB-706F3E01B4DD-low.png)

On the right of the toolbar, you will see a 'Search' option
to search keys from the list of all available keys for that application. You
can also view keys by different applications by clicking/expanding 'All Keys'.
Clicking only one application will list only the keys needed for that
application to run. By default, 'All Keys' will be selected. You can also
filter keys by Title or Value or you can uncheck the System Keys to remove them
from appearing in the Scenario Editor. See below the 'filter' options.

![](GUID-78BA8D83-4ACB-47A2-BF6B-62F9429DA905-low.jpg)
