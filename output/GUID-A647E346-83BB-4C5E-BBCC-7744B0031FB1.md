# DELTA

Type = Real(+)

Default = 2.0

Range = 1.0-999,999,999

The static penalty term multiplier Î´ as described by Equation 8. Setting
this value higher will result in enforcing the boundary constraints at a lower
number of iterations. The default value should be sufficient for most problems;
the range should be between 1.0 and 2.0.
