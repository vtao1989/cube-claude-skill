](GUID-283416D6-F32C-4DBC-AA88-096B276C3AFB-low.mp4)To dynamically visualize trip distribution patterns in a Map:

1. Right-click on Map Layers
    > Add Dynamic Zone
   Layer...
2. Select the polygon layer in
   the
   Data Source tab.
3. Select the appropriate field
   from those available from the loaded polygon in the Zone
   field drop-down
4. Select the matrix file
   containing the trip information that you wish to load into the Dynamic layer
5. Choose the appropriate table
   from the ones available in the
   Matrix table drop-down.
6. Select which value you wish
   the Dynamic Layer to show:
   1. From Selection (will treat the selected zone as the
      origin)
   2. To
      Selection(will treat the selected zone as the destination)
7. Click
   Apply

Select any of the zones from the boundary layer.  
![](GUID-D4B4D763-989A-4228-AB13-E3D2E67A30A5-low.png)

The selected zone will be outlined and the other zones will
be colored in a density based gradient of trips generated
from/to
 the selected zone.

Dynamic Zone layers will vary depending on the zone that
has been selected and can be customized:

1. Go to the Layer Properties
   window for the Dynamic Zone
2. Click on the
   Renderer
   tab.
     
   ![](GUID-4ECA6BCD-2D1C-46EF-AD19-52E432465E2C-low.png)
3. There are three symbols
   added by default: the selected zone symbol, the variable zone coloring symbol
   and the zone boundary symbol.
4. You can customize any of the
   symbols by right-clicking on the setting you wish to change.

For example:

1. Right-click on the second
   symbol to show the other available rendering options:
   1. Classbreaks:
      (Default) Will distribute the trips in a number of buckets with a specific
      value distribution
   2. Range: Will display
      the trips in a color gradient from the colors selected
   3. Dot
      Density: Will display the trip distribution density in colored dots.

To add labels to the zones:

1. Click on the
   Label
   tab
2. Right-click on the
   ABC
   symbol and select the variable you wish to display
3. Customize the label with the
   available options
