# Adding Two-Way/One-Way Links

[

In this tutorial, we will look at how to add a link
(two-way or one-way) to a CUBE highway network in the GIS window.

1. To create a two-way link, make sure Toolbar > Two-way
option is turned on.

![](GUID-C3BF4224-FA90-4922-8A7C-5DE5C8F93758-low.png)

2. Select the link dataset in the map table of contents to
which you would like to add the link.

![](GUID-9C292B2D-1651-4830-87B9-8F6D7A337EF0-low.png)

Select Toolbar > Create Links. The mouse cursor will
change to an edit cursor.

![](GUID-B8FF2F46-7434-4C75-BC18-3564C75158A3-low.png)

3. To create the link:

- Click on an existing node or
  a new location where the new link will start.
- If you are clicking on a new
  location, depending on the node numbering setting (automatic vs. manual), you
  might be prompted to enter a new node number for the start node. If you are
  prompted, please enter a new node number.
- Click on additional
  locations to add vertices to your link shape.
- You can finish sketching the
  new link and accept the updates in the following ways:
  - Double-click on the
    desired end of the link to accept the new link. This could be an existing node
    or a new location. In case of a new location, you might be prompted for new
    node number.
  - After clicking on the
    end node of the link, right-click and select Accept Sketch/Cancel Sketch.

![](GUID-D2353F46-197F-4DAF-9AEE-402F036BBC06-low.png)

- After clicking on the end
  node of the link, click Toolbar > Accept Sketch/Cancel Sketch.

![](GUID-17BA79B0-0488-4D3D-89E6-36C4C0ED4B65-low.png)

Tip: When in the link sketch mode,

- To pan: CTRL + left-click
  hold + drag
- To zoom out: CTRL +
  right-click hold + draw box
- To zoom in: Right-click
  hold + draw box

4. After the new link has been created, the link attributes
will be listed in the feature table.

![](GUID-FF8F9B99-975F-4A2C-AF0D-26C330CFE8BF-low.png)

5. To create a one-way link, turn off Toolbar > Two-way
editing option.

![](GUID-87E0352C-687A-43D5-9068-AD1F6B928996-low.png)
