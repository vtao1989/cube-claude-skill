# Add/Update Transit System Curves

In this tutorial, we will look at how to add and update the
transit system curves such as wait and crowding.

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-CCC97E19-2040-41EA-851F-D418DFA71B8C-low.mp4).

](GUID-CCC97E19-2040-41EA-851F-D418DFA71B8C-low.mp4)

1. Select the Public Transport (PT) dataset in the map table
of contents.

![](GUID-7A4878A4-E5B1-4032-BD38-D3FEAA37F8E9-low.png)

2. You can access the Transit System Data tool by:

- Right-clicking the PT
  dataset > Open Transit System Data.

![](GUID-4E6A4FC2-DA11-4F6C-A015-9E1FC57B5655-low.png)

3. Within the Transit Data Editor window, click on the Wait
Curves tab.

![](GUID-BC4DC000-243A-48ED-8CF2-BC4057D082EB-low.png)

To add a new wait curve > click on Add Curve.

![](GUID-FB2C3B61-BE43-404F-A6EB-A8C0E3C95AF4-low.png)

- You can add/update the
  attribute values of the new curve.
- Click on Save the new curve.

5. To create a new point in the curve > click on Add
Point.

![](GUID-9B5C6D8D-D750-46DB-887B-A04441609054-low.png)

- You can add/update the
  attribute values of the new point.
- Click Save to confirm the
  changes.

Tip: To move
the curve point manually, click/drag it in the graph to a new location. The
values for the Headway and Wait Time will update accordingly. This is also
applicable for crowd curve with its Utilization and Factor values.

6. Within the Transit System Data window, click on the
Multiplicative Crowd Curves tab.

![](GUID-C3B97993-51E3-4ABA-A654-6D17DCF2F6D8-low.png)

7. You can add/update a crowd curve by repeating step #4.

8. By repeating step #5 will create a new point for the
crowd curve.

![](GUID-2B1083C4-06ED-480E-A50E-D6CF883119A4-low.png)
