# Setup the environment with an external IDE

There are several valid options for setting up the
environment to work with Python and the CubePy package externally from the CUBE
Base environment. A possible option is to create a new environment using
Anaconda Navigator, installing all the needed packages.

Then, using the desired IDE, add the Conda environment as
interpreter, and the below folder to the interpreter paths for the selected
environment:

C:\Program Files\Bentley\CUBE7\Python
