1. Open Layer Properties window:

- Double-click on the layer
- Right-click on the
  layer > Properties

2. Node
tab > Chart
 tab

3. Click on Add Category to add a category based
on a variable or field:

a. Right-click/ click on the color box for the first
variable/field:

![](GUID-19E52FE2-7FEF-4A96-9D47-DD63645BC567-low.png)

i. Click on the Variable drop-down to display all the
available fields and variables (matrices and tables) that can be used for the
chart.  
![](GUID-9AB7C305-E7B3-405E-9E64-AC88430ED7EE-low.png)

The variable will now be displayed in the corresponding box:
  
![](GUID-7A8D21C6-AADC-482E-8E69-71CD574E7969-low.png)

ii. Click on the color box to Open the Select Color window
and pick a Color>
OK
  
![](GUID-7A18B7B7-B8C3-4169-806C-A3EB676C253E-low.png)

iii. Use the dragger next to the color box to adjust the
opacity.

iv. To submit changes, click on the following:
![](GUID-70A040D8-7108-4605-A96F-D8E6DD231257-low.png)

b. Click on the empty variable box to display the field
drop-down:  
![](GUID-AE7F5A44-EE11-412B-9196-4ABD3C8CD5E5-low.png)

i. Select the appropriate field from the list

ii. Click on the color box to change the color and opacity.

4. To add a New variable:

Click on New variable to select either:

a. From Matrix:

The Matrix variable Definition will open:

1. Type in the matrix name
2. Click on the browse symbol
   to Open File Explorer to look for the Matrix file
3. Click on the drop-down to
   select the appropriate table from matrix file.
4. Select which values from
   the matrix you want CUBE to use:
   1. Row
   2. Column
   3. Intrazonal
5. From the Method drop-down
   select the Method you want to apply:
   - Total
   - Minimum
   - Maximum
6. Click on
   Add

The matrix will be added below the toolbar. Double-click on
it to modify it.
  
![](GUID-2A2FD6A6-3258-47D1-88ED-46680F78565D-low.png)

b. From Table:

The Table Variable Definition window will open:

1. Type in the name to assign
   to the table.
2. Click on the browse symbol
   to Open File Explorer to look for the table file.
3. You can enter a query
   definition in the box if needed.
4. Select the appropriate Zone
   field from the list available under the drop-down.
5. Select the value field from
   the drop-down
6. Click
   Add

The table will be added below the toolbar. Double-click on
it to modify it.
  
![](GUID-5E6BC35D-CC4E-4A87-A3E8-319C0FC2DF36-low.png)

5. Click on Pie Chart to customize the
Chart:
  
![](GUID-C81634A1-DF8C-478C-97DB-53B4446F2B12-low.png)

- Select if you want a Pie or
  Bar Chart
    
  ![](GUID-3F5E3064-C04F-48E1-BA29-80CC85C111C0-low.png)  
  ![](GUID-F78CA581-2173-4146-9C08-22AF426CB4CE-low.png)
- If needed type a condition
  in the box and use the drop-down to display the available fields and variables.
    
  ![](GUID-61A4DA09-A4D5-447B-A65F-C72DDBF5D073-low.png)
- Use the size dragger to
  adjust the chart size.
- Click on the Relative size
  to adjust the size of the charts according to their values.
- Click on Scale to size the
  charts relative to the Zoom level.

6.
Click on the following symbol to display pre-defined color
palettes that can be applied for the charts:
  
![](GUID-3752FCAA-2D9B-4827-814C-6C015ECA59D3-low.png)
