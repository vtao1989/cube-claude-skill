# Add a Transit Line

In this tutorial, we will look at how to add a new transit
line.

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-9CB222DF-B908-4900-935C-09BC77B5C286-low.mp4).

](GUID-9CB222DF-B908-4900-935C-09BC77B5C286-low.mp4)

1. Select the Public Transport (PT) dataset in the map table
of contents.

![](GUID-4AD451D5-5091-4E6F-BB70-3A0F1E629950-low.png)

2. Right-click on the transit layer and select Properties.

![](GUID-ECB56AF8-3DAA-4657-B09C-D0D63E0233B9-low.png)

3. In the Data Source tab, enter the cost function you
would like to use for shortest path calculations between stop nodes. Click
Apply and close the layer properties window.

![](GUID-30DD8158-94CC-43E2-8774-257767B9AAD9-low.png)

4. Click on the Toolbar > Create Transit Line tool.

![](GUID-1CAC8908-B624-4DF9-B1D7-F4C4B7A2CAFE-low.png)

5. To create a new transit line:

- Click on an existing node to
  create the first transit stop.
- Continue clicking on the
  next stops in your new transit line, CUBE will auto route the line between
  stops along the shortest line.
- To create a direct link
  between stop nodes instead of the shortest path, use CTRL + click.
- Right-click the last node or
  anywhere in the map window > Accept Sketch.

![](GUID-9E2BCEE0-7CE9-4DCE-86B2-F30EC128481B-low.png)

6. Provide a name for the new transit line and assign a
transit mode.

![](GUID-0EC307F0-EF14-4D3A-B3BE-69AD600CFA86-low.png)

7. It is also possible to copy line attributes from an
existing line by turning on the copy attributes option and selecting a line
from the list. Click OK to confirm the settings for the new transit line.

![](GUID-7BE19D15-2AB0-48C7-8EC1-2D4E8039281C-low.png)
