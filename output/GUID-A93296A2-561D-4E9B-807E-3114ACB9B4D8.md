# NCLASSES

Type = Integer(+)

Default = 1

The number of user classes to be estimated. This value
determines how many input OD matrices the user must supply to the program be it
the number of matrix tabs (excluding confidence matrices) in the static matrix
file, or the total number of dynamic matrices. Note that if using the
[DYNCLASS](GUID-876F348E-7006-40A3-A8CD-7711B465AE97.html)
parameter to estimate a single dynamic user class from a problem with multiple
classes that the number of classes should be set to 1.
