# What is CubeAPI

The CubeAPI (Application Programming Interface) is written
in C++, it replicates and extend CUBE Voyager functionalities. The CubeAPI is
installed with the CUBE 7 installation.

The CubeAPI classes can be accessed directly from native
C++, and using different wrappers, permitting interoperability between systems,
however Python has become increasingly popular in the transport modelling
industry and was the first choice for supporting CUBE's API through the CubePy
package.

CubeAPI functionalities will be continuously extended, at
the current state of development it provides the following main components:

- [CUBE Network / CUBE Database API](GUID-1951253C-5924-4AC2-B96F-B84853A65A13.html#GUID-1951253C-5924-4AC2-B96F-B84853A65A13)
- [CUBE Matrix API](GUID-1951253C-5924-4AC2-B96F-B84853A65A13.html#GUID-1951253C-5924-4AC2-B96F-B84853A65A13)
