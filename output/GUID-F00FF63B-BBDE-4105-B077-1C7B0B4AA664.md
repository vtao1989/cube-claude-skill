# Python Installation

Python is freely available and distributable, and it is
part of the CUBE software installation in the folder:

C:\Program Files\Bentley\CUBE7\Python

Python can also be separately downloaded and installed from
the Python.org download page. At the current state of development, the
supported versions of Python with CubePy are 3.7, 3.8, 3.9, 3.10.
