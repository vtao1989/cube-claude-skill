# Avenue packet log files

Analyst Drive processes
Avenue packet log files to create the route choice
probability matrix, and also to produce the warm-up period OD matrix if using
the warm start option.
Analyst Drive converts the packet log file into a
readable data table from which route choice probabilities can be extracted.
