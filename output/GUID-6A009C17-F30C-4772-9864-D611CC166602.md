Note: CUBE does not allow
to write data with a different data type than that of the field.

After making changes you can:

- Click on the Commit Changes
  symbol
  ![](GUID-62DD1479-9731-4AF5-B31A-FA5F33C3002C-low.png)or
  click on the drop-down and select the option to save or commit to data changes
  in the file.  
  ![](GUID-42830CD9-5E57-46FA-B2FB-A38DDCAFFCF6-low.png)
- Click on the Revert Changes
  option to undo all the changes done since values were last saved.
    
  ![](GUID-1207081B-B4AF-4160-B27F-CD294D8E1C3C-low.png)  

  Note: Once you commit to
  changes you cannot revert or undo the action.
- Click on the redo symbol to
  go back to the last undone change
    
  ![](GUID-278C3CFD-4B60-4D18-91F8-CB242D83AE83-low.png)  

  Tip: Use the
  CTRL+ Y shortcut to redo
- Click on the undo symbol to
  go back to the last recorded change
    
  ![](GUID-A0E8D04A-832C-4E97-8435-5E6770EF0984-low.png)  

  Tip: Use the
  CTRL+Z
  shortcut to undo
- Click on the refresh symbol
  to reload the data from the disk.  
  ![](GUID-AFC12CD5-AFF5-4AF1-8382-2804954A3753-low.png)
