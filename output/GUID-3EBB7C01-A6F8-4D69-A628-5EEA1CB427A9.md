# Project Explorer Overview

Project Explorer corresponds to the left panel which is by
default, displayed when starting CUBE. This panel can be undocked, moved and
docked back to another location as well as hidden/unhidden from the View menu.

Project Explorer is home to individual managers who handle
most main components of a CUBE model, it is divided in two tabs which are :
Project and
Model.

Project:

- Scenario Manager

Model:

- [Application Manager](GUID-D2B8014C-2041-406E-9497-4F2DDDB9F16B.html)
- Key Manager
- Data Manager
- User Programs Manager

It is also possible to Customize the contents available in
Project Explorer through the Custom tab.

To access this option right-click on either Project/Model
tab and either select the Customize option or start selecting/unselecting the
items you wish to display. A custom tab will appear after the Model tab and
will include the select elements:

[Custom tab screenshot]

Project Explorer also contains a small toolbar which allows to perform
some operations in the project level like:

- Create a new project (see
  [Create/ Open a project](GUID-6E805840-77D4-4571-ACA5-F5A0BE002F12.html) for more detailed information)
- Open an existing project
- Import a CUBE 6 catalog (see
  [Create/ Open a project](GUID-6E805840-77D4-4571-ACA5-F5A0BE002F12.html) for more detailed information)
- Save the currently opened
  project
- Reload the current project
- Run the currently selected
  scenario and application
- Filter keys by color label
