Scopes allow users to have more flexibility when they want
to consolidate all files into one level. It allows the user to see file
linkages of the application in one location. In this tutorial, we will look at
how to add a new scope, adding files to the scope, and creating linkage lines
for the files within the scope.

1. To create a new scope, click Toolbar > Scopes >
click on the Â+Â symbol.

2. Give it a new name and click Add.

![](GUID-1D783BAD-9F8D-46BB-9DE2-00A41EC99E4B-low.png)

- You can also right-click in
  the Application Editor > Add > Scopes > click on the Â+Â symbol.
- Tip: You can
  also right-click any output file > Scopes > click on the Â+Â symbol. This
  shortcut summarizes steps #2-4.

3. Close the Scope Management window.

![](GUID-37751395-3C37-46AA-B7A9-EDB075541269-low.png)

4. To add an output file to a scope, right-click any output
file > Scopes > check the box for the new scope > close the window.

![](GUID-3415D10D-56C5-4A53-9A94-05480F85E7D7-low.png)

5. If the files are located elsewhere in the application, it
is possible to create linkages to those files. From the application screen >
access Scope > turn on the new scope.
