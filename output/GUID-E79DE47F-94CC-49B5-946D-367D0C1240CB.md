# Running from Application Manager

Once all data has been properly set up, the
Analyst Drive program can be executed by
double-clicking the
ANALYST\_DRIVE program box in Application Manager.
This will pop up the task manager along with a Windows command shell that will
produce streaming output of the estimation process. Most of this information
can simply be ignored by the user but may be useful. The shell will close after
program completion, but the user should check the output print file to ensure
that the program completed successfully (this will be stated at the end of the
file).
