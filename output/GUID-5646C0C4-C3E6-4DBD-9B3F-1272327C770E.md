# USECONF

Type = Logical

Default = F

This tells the program whether or not to actually use the
confidence matrices present in the matrix file. Note that all confidence values
given to the
Analyst Drive program are divided by 100. If
USECONF=F, all OD entry confidences are set to a
default value of 1 (=100/100). If
USECONF=T, the confidence matrix will be loaded
into memory as the value divided by 100. Note that
[CONFMAT](GUID-E195EAE8-D5ED-4873-AC1B-BF2F187D9C46.html)=T is required to use this option, if
[CONFMAT](GUID-E195EAE8-D5ED-4873-AC1B-BF2F187D9C46.html)=F the
USECONF value is ignored.
