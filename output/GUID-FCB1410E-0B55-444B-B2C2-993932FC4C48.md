# MPIWILD

Type = Character

Default = n/a

Allows the user to use a wildcard name in place of the drive
name. This is required for when using MPI on a distributed cluster which
utilizes seperate hard drives. For example, if you were running on a cluster
with nodes cluster1, cluster2, cluster3, and cluster4 where each has it's own C
drive and the input files for the
Analyst Drive program were kept on cluster1 then by
defining
MPIWILD=\\cluster1\c$ the other cluster nodes will
know that the file paths refer to the C drive of cluster1.
