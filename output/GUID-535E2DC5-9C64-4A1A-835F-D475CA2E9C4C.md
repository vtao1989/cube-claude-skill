# WARMAT

Type = Logical

Default = F

When using a warmup period, you may include the warm up
period matrix as the first tab of the input matrix file by setting
WARMAT=T. You do not need to use the warm up
period option if you have a warm up period matrix tab present, but you still
must set
WARMAT=T if the first matrix tab is reserved as a
warmup matrix. If
[WARMUP](GUID-E90B62F5-B190-40F2-ABC1-807D710D704F.html)=T and
WARMAT=F, then
Analyst Drive will automatically generate a warm up
period OD matrix from the
Avenue packet log file, and so it is not necessary to
provide in the matrix file unless so desired.
