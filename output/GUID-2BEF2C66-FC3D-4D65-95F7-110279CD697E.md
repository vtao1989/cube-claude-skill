# CUBE Cluster control statement summary

- DISTRIBUTE - Global control for
  DP options.
- DistributeMULTISTEP - Initiates
  MDP script block in Pilot unless the global switch is off.
- EndDistributeMULTISTEP - Ends a
  MDP script block in Pilot.
- Wait4Files - Sets a wait
  condition to insure all MDP steps of a block are completed prior to running
  subsequent model steps.

The following control statements are available in the CUBE
Voyager Matrix and Highway programs to implement IDP

- DistributeINTRASTEP
  - Initiates IDP of the current Matrix or Highway step.
