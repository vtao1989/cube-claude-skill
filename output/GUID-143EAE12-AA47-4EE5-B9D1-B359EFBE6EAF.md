# Input files for dynamic estimation

For dynamic estimation, the user has the option of using Cube Avenue or
another assignment method. The default option easily allows users to input
files used in conjunction with Cube Avenue into the Analyst Drive program. When
using the
[GENERAL](GUID-537E9D14-8742-4648-BCD5-93CC253CD77F.html)
option, users input specially formatted text files that are
platform-independent.
