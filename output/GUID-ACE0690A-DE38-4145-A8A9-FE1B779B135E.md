The scroll lock tool in CUBE permits the user to
synchronize multiple tabs whether it be text files or networks. Applying scroll
locks lets you synchronize horizontal and vertical scrolling for tables/text
files and pan/zooming for map windows.

1. To synchronize multiple
   tabs, turn on the scroll locks inside each of the tab windows you would like to
   synchronize.  
   ![](GUID-984EBFA4-DB58-41BB-A467-CA9CF98D50D7-low.png)  

   You can position the windows as needed for comparing the
   data in the synchronized window side by side.
