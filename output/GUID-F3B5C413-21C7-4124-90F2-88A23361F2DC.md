For matrix tables you can add and view zonal references instead of the
zone numbers. This will help the user with easier identification of zones in
their model region.

To add a new zonal reference:

1. Right-click on any
   column > Zone
   labels
2. Click on
   New Zonal Reference  
   ![](GUID-E46A9A8B-5335-4999-9772-F99C906C757F-low.png)
3. Rename the new zonal
   reference if desired by clicking on it and typing the new name.
4. Click on the column or row
   header to insert the zone names for each of the zones you wish to
   identify.  
   ![](GUID-E40FCE0B-82B0-430B-A600-0DD5AA8F73BC-low.png)
5. Save the matrix file to save
   the zonal reference.

To select a zonal reference to display:

Right click on any column
 > Zone
Labels:

- Click on the check mark next
  to zonal references you wish to display, if any.
    
  ![](GUID-181F838B-8A18-47E3-9FFA-DFAF19E4F2FF-low.png)
- Use the
  Filter.. section to look for a specific zone
  reference.
- Use
  Clear selection to uncheck the current zonal
  reference.
- Click on the X next to the
  reference zone to remove it.
    
  ![](GUID-D4BEB462-9115-4B11-89B1-EA9D7EEB9529-low.png)
