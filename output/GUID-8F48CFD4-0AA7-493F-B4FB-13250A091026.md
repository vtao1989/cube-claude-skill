# General dynamic estimation

The
[GENERAL](GUID-537E9D14-8742-4648-BCD5-93CC253CD77F.html)
option for dynamic estimation is available for users who wish to perform
dynamic estimation using an assignment program that is an alternative to
Cube Avenue. This option uses text file inputs for the
route choice and OD matrices, and the dynamic counts. The general input text
files can be comma or space delimited; they just need to follow the proper
order of the format provided in the examples given in the figures here. As of
the first release of
Analyst Drive, only a single user class may be
estimated at once when using the
[GENERAL](GUID-537E9D14-8742-4648-BCD5-93CC253CD77F.html)=T
option. General dynamic input files are declared in a special control file
which is itself a file input for the
Analyst Drive input file. The GEN DYN FILES format is
described in more detail later in
[Chapter 5,
ÂDynamic EstimationÂ](GUID-E65C2842-16A4-4A3A-A58C-A37C93E42EBD.html), and an example of this file is found there in
Figure 16.
