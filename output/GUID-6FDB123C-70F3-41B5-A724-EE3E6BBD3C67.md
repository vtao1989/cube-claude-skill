# Run Application Sub-Group

For this exercise, we will learn how to run an application's
sub-group. These abbreviated runs are useful for many reasons. If a model fails
in a specific location, a short run will save time and effort to isolate the
cause. Some sub-groups are specialized where they do not require a full model
run, e.g. sub-area analysis, scenario comparisons, etc.

1. Open any project file which already contains groups.

![](GUID-0E55CF85-7776-4C07-AD6A-9B7DA3337E9F-low.png)

2. To run a specific group, right-click it > Run.

![](GUID-5D8EFAEA-CFAC-4698-A23A-36E46E19DD99-low.png)

3. Click Run on the Run Application window to initiate the
process.

4. The run should be completed shortly. You can close the
window. In this exercise, we will not cover the other options of the
Application Run Monitor.

![](GUID-60119748-5F25-4B73-87F9-6255F17FDB6D-low.png)
