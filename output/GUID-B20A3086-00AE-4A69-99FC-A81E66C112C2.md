# Overview

Dynamic Estimation is available in
Analyst Drive to estimate matrices over multiple time
periods. To perform dynamic estimation, the required inputs are OD matrices for
each time period, count files corresponding to OD matrices for each time
period, and a dynamic route choice probability matrix.
