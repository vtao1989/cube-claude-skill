It is possible to retrieve cells from a table in CUBE and
paste them in another document.

1. Select the cells you want
   to copy
2. To copy:
   - Click on the Copy
     symbol on the toolbar  
     ![](GUID-57EFF3A7-9F28-47E6-8D1F-A0433437808A-low.png)
   - Right-click anywhere
     inside the
     selection > Copy
   - Right-click on any
     column/field
     name > Copy

     Tip: You
     can copy using the shortcut
     CTRL+C
3. You can use the CTRL+V
   command or right-click
    > Paste
   to insert the copied data in the desired
   location.

Note: The corresponding
column headers for the copied cells will be also copied.
