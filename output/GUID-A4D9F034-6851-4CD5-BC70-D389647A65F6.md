- Create a new application:
  - Go to
    File > New > Application

    Tip: You
    can also use the keyboard shortcut
    CTRL+ALT+A
  - In Project Explorer, in
    the Model tab go to Application Manager.
    Right click
     > New Application
- Adding an existing
  application to a project:

1. Right-click on the
   Application Manager section
    > Add
   Existing..
2. Browse on the file directory
   for the
   Application > Open

- Import a CUBE 6
  Application:

1. Right-click on the
   Application Manager section
    > Import
2. Browse on the file
   directory for the CUBE 6
   Application > Open > Enter

- Opening an application:
  - Double-clicking on the
    application
  - Right-click
     > Open.
