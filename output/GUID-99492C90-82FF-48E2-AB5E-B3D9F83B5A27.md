# Keys Manager

Keys define items that vary each time an application in a
project is run. They may represent a file, number, character string, or Boolean
flag. They are assigned values by scenario.

All the available keys in a project, system and application
keys, are listed in the Key Manager section in the model tab in Project
Explorer. The Key Manager section allows to organize existing keys under user
created categories or colors, as well as add, remove, duplicate and rename
those categories and keys.  
![](GUID-84D95FBA-34A3-45A6-A66D-230405BE77C8-low.png)

Tip: Hovering
over any of the listed keys will display a summary of the main details of the
key.

You can edit a Key's definition by opening the Key
Definition Editor, which can be accessed through the Key Manager context menu
or by double-clicking on the desired key. (see
[Editing keys](GUID-EA8AAD59-C7E2-4A6B-96DB-09651A41306B.html)for more information about the Key Definition Editor and check the
tutorial
[Working with Scenario keys](GUID-55072C8F-26B4-49C5-A6F4-F85726B3D64B.html) for information on how to set up
your keys)
