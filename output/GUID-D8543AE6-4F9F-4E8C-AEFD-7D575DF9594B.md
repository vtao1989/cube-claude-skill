# Adding Nodes/Centroids

In this tutorial, we will look at how to add a node and
centroid to a CUBE highway network in the GIS window.

1. To create a node, make sure Toolbar > Node option is
turned on.

![](GUID-3ED94193-D49A-41FA-B1BF-56BB9883E581-low.png)

- You can also create a node
  by right-click anywhere inside the GIS map > click on Node.

![](GUID-520202A6-FA63-4AE8-AD8F-4B06BE4982B1-low.png)

2. The Create Network Node window should appear. Make sure
Nodes is checked in the lower-right. In the upper-left box, enter a value that
satisfies from the Available Nodes criteria.

![](GUID-BEE0FF24-4CD2-432D-9077-9E30044F60A6-low.png)

- You can enter/select a value
  from Toolbar > drop-down arrow of Node Numbering.

- Tip: From
  the same drop-down option > click on the Ââ¦Â icon next to the Next N box
  will open the Next Node window.

![](GUID-E4B4DA54-96D4-4B0B-A3A1-8E870CA94D45-low.png)

3. By default, the Automatic Node Numbers option is turned
on. There is a box surrounding the Toolbar > Node Numbering option. By
checking the box, CUBE will disable the automatic node numbering option. Turn
it off for now and repeat step #1. When you create a new node with the box
checked in Create Network Node window, the box will appear around the Toolbar
option.

![](GUID-4D8FD00D-F9C9-4E7C-80E3-81A01B8F2D1D-low.png)

![](GUID-FF9B2FCC-FF12-4AA2-B893-EA38EFBC0317-low.png)

4. To create a centroid > turn off the Toolbar > Node
Numbering option.

5. Repeat step #1.

6. At the window again, check off the Nodes and check the
box for Centroids.

![](GUID-32B25DF7-98A1-49F9-B394-CB7D678D59C9-low.png)

- Tip: You can
  also switch to Centroids by Toolbar > drop-down arrow of Node Numbering >
  Centroids.

7. Enter a value that satisfies from the Available Nodes
> click Create.

![](GUID-0F14496E-CA7F-49C6-8A2F-59DEF3FEC4D1-low.png)
