# AUTOTE

Type = Logical

Default = F (see note below)

The
AUTOTE option allows user to use trip end data
that is generated directly from the static input matrix file. With the
AUTOTE option, no user generated trip end files
are necessary to incorporate trip end data as the program can automatically
generate this data by adding the individual rows and columns from the input OD
matrix. Use
AUTOTE=T to automatically generate trip end data
for use with the
[TRIPENDS](GUID-6043C302-4754-43B6-B9DB-A3832D5E59D4.html#GUID-6043C302-4754-43B6-B9DB-A3832D5E59D4)
option. Default value = F, however if
[TRIPENDS](GUID-21AA278C-ACE1-4CD2-9FEB-AF556603DDA8.html)=T
and
FILETE is not selected then this value will be
used by default.

Note that if trip ends are selected in input, but they need
to be excluded from the Matrix Estimation process, all three parameters need to
be False:
TRIPENDS=F, FILETE=F, AUTOTE=F.
