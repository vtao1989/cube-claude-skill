](GUID-54B6004B-CF97-447D-9333-B8258BE3D304-low.mp4)To create a CUBE project from scratch:

1. Go to
    > File > New
    > Project

   Note: The
   CTRL+ALT+P
   shortcut will open the New Project window.
2. Write down the name of the
   project
3. Browse for the location
   where you wish to store the project
4. Select if you wish CUBE to
   create a subdirectory for the new project
5. Select if you wish CUBE to
   generate a README.md (Markdown file) for you
6. Click on
   Create
     
   ![](GUID-956D375A-39B8-4F01-B480-A20FAFA43A14-low.png)
