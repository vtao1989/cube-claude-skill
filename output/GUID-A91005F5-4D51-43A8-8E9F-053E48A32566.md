# Using Edit History for Undo/Redo

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-FF13844C-E7DF-48E8-8434-2B351FB3F7D2-low.mp4).

In this tutorial, we will look at how to use the History option to
undo/redo edits which effects the application flow chart.

1. Below is an example application.

![](GUID-FEFF88FE-2FD1-4A4D-8040-D6978BA87730-low.png)

2. Make some changes that would impact the application's workflow, e.g.
delete a program, group, add a new item, etc.

![](GUID-C6D7F0D2-FEF5-4C88-B93C-9A79BE2E5C67-low.png)

3. Click on Toolbar > Tools drop-down arrow > History.

![](GUID-25ABA0FE-3436-4E71-BF75-E52278384C15-low.png)

- You can also right-click
  anywhere in the Application Editor > View > History.

4. By clicking the last item on the list, it undoes only that item.
Clicking on an intermediate item will undo that item and any items below it.
