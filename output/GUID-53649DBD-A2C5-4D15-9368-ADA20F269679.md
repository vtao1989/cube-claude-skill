Application Manager is the Application panel in Project
Explorer, where the user can manage different applications and its groups.

![](GUID-D45EBE9D-A6F0-4926-AE05-434292D38AAA-low.png)

Use Application Manager to:

- Drag and drop any
  application to move it inside AM.

- Expand and Collapse
  Application groups:
  - Click on the tree node
    (small rectangle) on the left of the Application name to expand(unfilled) and
    collapse(filled) the groups of the selected application.
  - Right-click on an
    application and select either the
    Expand All
    or
    Collapse All options
      
    ![](GUID-D3EF0BAC-9B27-496E-BD08-1F8203F40B82-low.png)
- Create a new application:
  - Go to
    File > New > Application

    Tip: You
    can also use the keyboard shortcut
    CTRL+ALT+A
  - In Project Explorer, in
    the Model tab go to Application Manager.
    Right click > New
    Application
- Once an application has been
  created, the user can open it by either:
  - Double-clicking on the
    application
  - Right-click > Open.

- Add an existing application
  to a project:

1. Right-click on the
   Application Manager section
    > Add
   Existing..
2. Browse on the file
   directory for the
   Application > Open

   Tip:
   Applications are stored under the Applications folder of a CUBE-project.

- Import a CUBE 6
  Application:

1. Right-click on
   Application
   Manager > Import
2. Browse on the file
   directory for the CUBE 6
   Application > Open > Enter

- Rename an application:
  - Click on the
    application name and rename it.
  - Right-click on the
    application > Rename
- Delete an application:

1. Right-click on the
   application > Remove
   .

   Tip: As a
   shortcut hit the
   DEL key to remove it.
2. Select the instances of the
   application to be removed in the dialog box:  
   ![](GUID-1D9613B9-129C-4B3A-A9A3-2CEADF84F368-low.png)
3. By selecting the
   Application box you will be removing the
   application only from the tree, while when selecting the
   Files box the Application is removed from the
   disk.
4. Click on
   Delete

- Create a copy of an
  application:
  - Right-click on the
    Application > 
    Duplicate.

  Tip: As a
  shortcut use
  CTRL+D to duplicate an application.

  The duplicated version of the application will be named after
  the original one and followed by an index describing the order of
  generation.  
  ![](GUID-6A7BB3CA-E23C-48B5-A05E-04DC9D94214F-low.png)
- Explore the file directory:
  - Right-click on an
    application
     > Explore..to
    open the File Explorer window showing the exact location of the application.
- Access the application
  properties:
  - Right-click on the
    application name
     > Properties.
  - The properties window
    will shows the details of the application selected including name and location.
      
    ![](GUID-0B58334C-58D5-4864-90CE-7F36F2E4CA1C-low.png)
