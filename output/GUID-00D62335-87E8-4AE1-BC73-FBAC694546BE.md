# Overview

We have developed a data assimilation type OD estimation
model that solves an upper level optimization problem that follows from a lower
level simulation problem. The bi-level scheme iterates between optimizing a set
of OD flows using a fixed trip proportion matrix, and between using
Highway or
Avenue for simulating route choice assignments. A flow
chart depicting this scheme is given in Figure 1 below
