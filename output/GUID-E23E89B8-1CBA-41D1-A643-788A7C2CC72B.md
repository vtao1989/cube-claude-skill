# NCORES

Type = Integer(+)

Default = 1

The number of shared memory cores the program will use in the
estimation. This can be as high as the number of shared memory cores on the
system (physical and virtual). If there are not enough cores available to the
program as requested through the
NCORES parameter, the maximum number that are
available will be used. Note that using the
NCORES parameter along with the [MPICORES](GUID-539DD601-CF95-495D-8993-D058970BF046.html) parameter the
maximum number of cores that may be in use is equal to
NCORES x
[MPICORES](GUID-539DD601-CF95-495D-8993-D058970BF046.html).
