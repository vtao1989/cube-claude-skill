# What is the Toolbox?

The CubePy Toolbox provides several ready to run tools to execute data
processing, analysis, and modelling tasks. These tools are providing a GUI for
the user to select inputs, outputs and parameters and run the tools
interactively.

It is also possible for power users to create their own Python tools and
import them in the Toolbox, to be executed via GUI from other users.
