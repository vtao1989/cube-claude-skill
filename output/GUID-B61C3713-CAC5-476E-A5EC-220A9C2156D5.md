# Compute Link/Node Attribute Values

In this tutorial, we will look at how to compute the attribute values of
a node and link.

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-D38BD5EF-2F8C-467E-A238-128CCEE32711-low.mp4).

](GUID-D38BD5EF-2F8C-467E-A238-128CCEE32711-low.mp4)

1. Right-click the highway link dataset in the map table of contents
> Open as Table. This will open the attribute table in the table editor.

![](GUID-80083ECE-1F8A-4E6D-B3AF-CA738E3E5DEB-low.png)

2. Click on the Compute tool in the table editor.

![](GUID-82E0FB4C-F0FF-410E-9799-0FDFEDB61AC0-low.png)

3. With this tool, you can compute any attribute values and apply the
necessary changes to the selected link or node dataset. For example, we want to
decrease the speed from 25 to 20 for all links with a functional class of 10.

![](GUID-D04B7C25-8FBC-476F-88CC-3347DE38D4B1-low.png)

![](GUID-4D965123-DE0F-49DE-9491-F0104826EF13-low.png)

Tip: To clear all conditions in the compute box, click on the
drop-down arrow next to Apply > Clear.

4. Click Apply to confirm the computation > click anywhere outside
the compute box.

![](GUID-B892A9DE-9051-44A7-983B-611452945CD4-low.png)

5. Repeat steps #1-4 for the node dataset.

Tip: When you begin to type down the name of the attribute in
the compute box, CUBE will provide a list of attributes with the same match.
You can complete the search by clicking on the desired attribute name.
