To display table statistics like minimums, maximums,
averages, etc. :

1. Click on
   toolbar > 
   Statistics  
   ![](GUID-D52565FD-B892-4867-8353-4AC3D5175067-low.png)
2. The Table Statistics window
   will open:  
   ![](GUID-D53E5E0A-5FCC-4795-BB6E-D36CC835B8F1-low.png)

Matrix Editor also has a similar statistics tool
![](GUID-03E7ECFE-8006-49D6-93E7-3F743C114531-low.png) as Table Editor, which allows to
visualize data metrics like:

- Zone Number: Displayed next
  to Matrix Statistics
- Statistics: Row number,
  column number and cell number of both selected data and the entire table.
- Other Data: Sum, average,
  minimum, maximum, non-zeros, non-zero min and non-zero average as well as intra
  zonal information, for both selected data and the entire matrix.

  to only display specific field statistics.
- Select > CTRL+C
  to copy statistics.
