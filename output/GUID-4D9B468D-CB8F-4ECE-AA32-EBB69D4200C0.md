# TRWGHTJ

Type=Real(+)

Default = 100

Range = 0-10,000

Weighting factor on the J trips for when using trip ends, use can be
viewed as similar to that of
[MATWGHT](GUID-03AB6DDA-D71E-4D9E-B6C3-0A7785770802.html).
The higher the weighting factor, the closer the J trip end totals of the final
adjusted matrix will come to matching those of the input matrix.
