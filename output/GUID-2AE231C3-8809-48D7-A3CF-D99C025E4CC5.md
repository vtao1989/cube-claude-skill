# KEEPFILES

Type = Logical

Default = F

When selected, the
KEEPFILES option will not delete the intermediate
output files created from processing the packet log file when performing a
dynamic estimation in conjunction with
Cube Avenue. These files can be used to perform a
[GENERAL](GUID-537E9D14-8742-4648-BCD5-93CC253CD77F.html) dynamic estimation which avoids the costly packet
log file processing step. This could be beneficial if the user simply wishes to
test/tweak some program parameters to see the effect on program output while
skipping the time consuming packet log processing phase of the estimation.
