# XLOBND

Type = Real(+)

Default = 0.1

Range = 0.0-1.0

Example: If
XLOBND=0.01 then the smallest value any individual
OD matrix entry could obtain during the estimation would be 0.01 times the
original value.

X lower bound, a scalar multiplier which defines the lower bound on the
individual entries of the initial OD matrix. When defining the matrix bounds, a
wider range will allow for a greater reduction in the cost function in each
estimation run, but will allow the OD matrix values to stray farther from the
original value. If the user does not wish OD entries to be zeroed out during
the estimation process, the lower bound must be greater than 0.0. If the user
does not wish any OD entries to decrease in value, then lower bound should be
assigned the maximum value of 1.0.
