# XUPBND

Type = Real(+)

Default = 10.0

Range = 1.0-1000

Example: If
XUPBND=10.0 then the largest value any individual
OD matrix entry could obtain during the estimation would be 10.0 times the
original value.

X upper bound, a scalar multiplier which defines the upper bound on the
initial OD matrix. When defining the matrix bounds, a wider range will allow
for a greater reduction in the cost function in each estimation run, but will
allow the OD matrix values to stray farther from the original value. If the
user does not wish to have any OD entries increase in value, then the upper
bound should be set to the minimum value of 1.0.
