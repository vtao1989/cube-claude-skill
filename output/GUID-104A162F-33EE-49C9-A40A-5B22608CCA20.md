# Running a Model

Models can be run in CUBE 7 using multiple ways. These are
described below:

Running Multiple Scenarios

Running more than one scenario at once is possible through
the Batch Run Composer which can be accessed through:

- Run button in Project
  Explorer
- Run button in Scenario
  Manager
- Run button in Application
  Editor

![](GUID-2CECCD06-094D-42A3-8B09-F4678237FEA1-low.png)

The Batch Run Composer will list all the Applications and
Scenarios available in the project, which can be selected to be run.

See more in
[Schedule Multiple
Scenarios for Model Run](GUID-658D8EE7-0DFA-448E-BA95-641B7E95CDC0.html)

Running an Application

To run an application in a Project.
Right-click anywhere in the Application > Application >
Run.

![](GUID-B8E2C4D2-2C94-4C66-8B53-D3FF6FF3637A-low.jpg)

Clicking
Run will launch 'Batch Run
Composer' dialog box with the application selected and the list of
available scenarios for the run.

It is also possible to run an application subgroup or a
group, see
[Run Application
Subgroup](GUID-6FDB123C-70F3-41B5-A724-EE3E6BBD3C67.html)

![](GUID-546AAFC4-CD4B-4FBE-8EFD-22B1561C7B2D-low.jpg)

Running a Program

To run a single program, double-click on the program and
the
Run Application dialog box will appear

-
Explore: Clicking this option will take the
location where .S file or the job script is created

-
Edit: Clicking this option will let the users edit
the .S file

-
Run: Clicking Run will launch
[Application Run Monitor](GUID-A55F683F-6A5F-4957-A0A1-D65FAD070C31) to run the model

Running a Group

To run a Group, right-click on Group, click 'Run' and the
Run Application dialog box will appear as above.
Clicking Run will launch the Application Run Monitor and run the Group.

![](GUID-A2A95E2D-A070-4026-B796-6AA0A840ECFA-low.jpg)

Running a Block

To run a Block, right-click on Block, click 'Run Contents'
and the
Run Application dialog box will appear as above.
Clicking Run will launch the Application Run Monitor and run the contents of
the block.

![](GUID-3F2F9FF4-858D-4341-A786-5046E311C627-low.jpg)
