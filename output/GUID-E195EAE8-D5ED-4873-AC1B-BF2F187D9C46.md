# CONFMAT

Type = Logical

Default = F

This tells the Analyst Drive program
whether or not there are confidence matrices present in the input matrix file.
CONFMAT=F if there are no confidence matrices
present with the OD matrices in the input matrix file,
CONFMAT=T if there are confidence matrices for each OD
corresponding OD matrix present in the input matrix file.
