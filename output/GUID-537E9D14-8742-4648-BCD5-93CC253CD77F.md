# GENERAL

Type = Logical

Default = F

This option is for use with dynamic estimation and signifies that the
user wishes to use general-format text files to perform the optimization. In
this case the user must provide the three general format files for dynamic
counts, dynamic OD matrix, and dynamic route choice probability matrix. In
addition, the user must provide the number of screenlines and time intervals
for the problem. General-format warm up route choice and OD matrices can also
be optionally input into the program when doing a warm start.
