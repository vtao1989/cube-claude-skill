# Building Bandwidth Maps

In this tutorial, we will look at how to create bandwidth
maps.

1. Select the highway link dataset in the map table of
contents.

![](GUID-1D01B8C5-794D-4EE0-8416-5716F805258B-low.png)

2. You can access the Layer Properties by:

- Right-clicking the highway
  link dataset in the map table of contents > Properties.

![](GUID-892C832B-A926-4597-9129-2908DD53C206-low.png)

- You can also access the
  Layer Properties by double-clicking on the dataset itself.

3. Confirm you are in the Link and Render tabs. Click on
the Add Symbol.

![](GUID-396F5E63-35AA-4ED5-982B-1CB5B2512C92-low.png)

4. Click on its width to access the Line Width window.

![](GUID-E3520FBC-170E-47D4-81A6-A213396977DA-low.png)

5. There are two options for creating bandwidth maps: width
by classbreak and width by range. Select the middle option first (width by
classbreak).

![](GUID-BF955EEC-0889-4CEB-9F62-8592890F7C7D-low.png)

6. Either enter a variable or click the drop-down variable
box to use it for the bandwidth map.

![](GUID-F1F42550-75D5-4BE8-BE5F-71B5B563F807-low.png)

Note: The Alignment
determines the position of the bandwidth for each link. For example, if the
left alignment was selected, the bandwidth will be thicker on the left side of
the link.

7. The number of buckets and value distribution determines
the total number of class buckets that will be distributed amongst the
bandwidth map. There are five distributions to select from: Custom, Equal,
Natural (Jenks), Quantile, and Custom.

![](GUID-B64B95CB-BA6C-4875-8AB7-6E0FD378378C-low.png)

Tip: It is
possible to select from several width template by clicking on its icon. These
options determine the thickness of the bandwidths based on ascending or
descending range.
