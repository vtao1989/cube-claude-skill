# Delete Non-Transit Legs

[

In this tutorial, we will look at how to delete non-transit
legs.

1. Select the Non-Transit network dataset in the map table
of contents.

![](GUID-DEBBE89D-F4EC-4937-BF6E-CBCE7921C3FB-low.png)

2. You can access the can access the Non-Transit Editor tool
by:

- Right-clicking the
  Non-Transit dataset >
  Open Non-Transit Editor.

![](GUID-4E28800D-BBAC-40CE-8E14-A7597F4114DA-low.png)

- If you have the dataset's
  attribute table open in the Feature Table, it is possible to access the
  non-transit editor by clicking on its icon.

![](GUID-52B8E008-A4CE-4991-BDE7-24B88A1C310B-low.png)

Tip: If you
selected a leg, it will show up in the Feature Table by itself. By accessing
the non-transit data editor, CUBE will automatically highlight the selected
leg.

Tip: You can
filter a non-transit leg by its name using the
Filter box of the non-transit editor window.

![](GUID-86C24317-2BF4-4180-960E-05A0CDB74A1C-low.png)

3. Select the non-transit leg >
Remove Leg.

![](GUID-11F5ED8B-92AE-4D37-BE92-49DE225DD526-low.png)

4. Click
Save to confirm the changes.
