# AGGINTRV

Type = Integer(+)

Default = 1

The number of dynamic time intervals to be aggregated from
a given
Avenue
assignment when the number of periodic counts is fewer than the
number of time periods used in the assignment. For example, if there are 18
time periods of 1 hour each present in the
Avenue assignment, but only 6 of those periods (1
every 3 hours) has count data, then
AGGINTRV
should be set to 3 to aggregate the sets of 3 individual hour
long time intervals into a single time interval in the estimation.
