# Update Non-Transit Leg Attributes

[

In this tutorial, we will look at how to update non-transit
leg attributes.

1. Select the Non-Transit network dataset in the map table
of contents.

![](GUID-ADE76D6E-C854-43E2-985B-ACC621D0C7D6-low.png)

2. You can access the Non-Transit Data Editor tool by:

- Right-clicking the
  Non-transit dataset in the map table of contents >
  Open Non-Transit Editor.

![](GUID-D11EEACD-EF34-4478-814D-2B7C31DD51FE-low.png)

- If you have the dataset's
  attribute table open in the Feature Table, it is possible to access the
  non-transit editor by clicking on its icon.

![](GUID-64E8DF02-79EC-4F8D-999D-9DD0478F550E-low.png)

3. In here, you can update any desired attributes of a
specific non-transit leg. Simply select a non-transit leg and change the
attribute values.

![](GUID-04B1DE58-A426-4C8E-BCC0-099C0D09A2A2-low.png)

Tip: You can
filter a non-transit leg by its name using the
Filter box of the non-transit editor window.

5. Click
Save to confirm the changes.
