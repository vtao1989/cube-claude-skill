# CNTCONF

Type = Logical

Default = F

If this option is selected, the program will use the count confidence
values in the estimation. The confidence value for screenline number k is
1.0+SUM(CONF(k))/100.0 where the sum is over all identical screenline numbers
in the screenline file. When using this option, the estimation will be biased
towards matching counts with high confidence values. Thus, this option may not
be desirable in all cases.

If this option is not selected, then all count values will be given a
confidence value of 1.0.
