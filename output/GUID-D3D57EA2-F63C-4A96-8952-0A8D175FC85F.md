# Create group from block

[

To help organize and ease the workflow of some models it is
possible to create a group from an existing block.  
![](GUID-06698E45-A392-44F2-900C-2F6239F40353-low.png)

1. Right-click on a populated
   block
    >  Convert to
   Group  
   ![](GUID-63AE1111-829E-4D9E-BBCB-FC278A0DCF82-low.png)
2. Name the new group in the
   Add Group
   window > Add
     
   ![](GUID-5078CC70-997F-4F2A-8842-C456E82EB8EF-low.png)

A group box will be added to either the application or group
where the block used to be in, and will contain all the contents of the
block  
![](GUID-3505B5A0-CC8F-493D-9E76-1F8CC7761DB5-low.png)  
  
![](GUID-DA213E40-A9D0-4065-8B18-31FD0C8E4DD6-low.png)

The new group will be added to the Application tree in
Project Explorer, and will reflect the new structure:

  
![](GUID-E6E7A9B3-1956-4B68-999D-09F70DCA9BC7-low.png)
