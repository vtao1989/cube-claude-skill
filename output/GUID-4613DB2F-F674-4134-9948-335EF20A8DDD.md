To visualize value ranges in a table:

1. Click on the drop-down in
   Scroll Lock
    > Value
   Gradient
     
   ![](GUID-4FE525F3-9E9E-4802-AD0F-95771EDDB747-low.png)
2. Click on the color box to
   open the Select Color window
3. Select the desired color
   for minimum and maximum
    > OK
4. Drag the slider next to the
   color box to modify the opacity of the color selected.
5. Modify the minimum and
   maximum values:
   1. Click on the box and
      type the new value ranges
   2. Use the up/down arrow to
      modify the value ranges.

Tip: It is
also possible to modify the colors in Table Options
