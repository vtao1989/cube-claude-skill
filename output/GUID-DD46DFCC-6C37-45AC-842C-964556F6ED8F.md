# WORKDIR

Type = Character

Default = n/a

A working directory must be provided to the program via the
WORKDIR
parameter; it can simply be linked to the SCENARIO\_DIR
catalog key using the
Link to Catalog Key menu option at the bottom left
corner of the parameters menu in the control file editor, or can be any
directory of the userâs choosing which has file write access. Intermediate
files produced by
Analyst Drive will be stored in this directory, and
those which are not deleted will be overwritten every time the program is run.
This parameter has no default value and must be defined.
