# WARMCNT

Type = Logical

Default = F

The
WARMCNT option is for estimating dynamic warm up
period matrices along with the regular time period matrices when performing
dynamic estimation. If
WARMCNT=T then the input dynamic matrix file must
be the same as the one input into the
AVENUE assignment and dynamic counts must be included
for each warm up period just as with the regular time period matrices. If the
program detects
WARMCNT=T, it automatically sets
WARMUP=F and
WARMAT=F regardless of their values.
