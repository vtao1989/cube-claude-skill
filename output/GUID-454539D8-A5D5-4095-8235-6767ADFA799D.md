# File Browser

File Browser introduces a familiar setting to visualize and
manage your projects. File Browser resembles Windows File Explorer and supports
the transfer of data between both.

File Browser consists of three panels:

1. The right-most panel will show the projects contents in a
tree-structure:

Folders can be collapsed/expanded and when selected will be opened in
the main panel.

2. The main panel will show the selected folder's contents:

There are 2 different view options:

- List view: Will show the
  files names, modification and creation dates and its sizes.
- Grid view: Will show the
  different files and folders in boxes with its corresponding icons and names.

Filtering files by their extension is also available in this section as
well as searching for a specific file

A user can also add new files folder and databases in this section.

3. The left section will show the selected file details like its path,
modification and creation date and size.

An additional dynamic tab will appear when a database file (.sqlite,
.gdb, etc) is selected. This tab will list the database contents. It is
possible to open the datasets as well as create new networks and datasets.
