# Delete a Link/Node

[

In this tutorial, we will look at how to delete a link and
node in a CUBE highway network.

1. Select the link dataset in the map table of contents to
which you would like to delete the link.

![](GUID-C68BA05D-5B03-4885-9D95-C583202EF9A0-low.png)

2. To delete a link:

- Make sure Toolbar .>
  Delete Features option is turned on.

![](GUID-1C32D63A-5987-4D17-AD0A-B84365BFC03F-low.png)

- Click on any link to delete
  it.
- You can also right-click on
  any part of a link > Delete > click on the desired link to delete it.

![](GUID-44F0D90C-82FE-4640-9F1D-422DE6D9F8A3-low.png)

3. To delete a node, select the node dataset in the map
table of contents.

![](GUID-A4A690CA-C86A-4433-969E-E7C90509673D-low.png)

4. Repeat step #2 to delete a node.
