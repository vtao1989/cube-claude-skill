In this tutorial, we will look at how to split a link in a
CUBE highway network.

1. Select the link dataset in the map table of contents
before you split a link.

![](GUID-2D26DD83-B6AE-4716-BDC1-DCEDFA1EA2CB-low.png)

2. To split a link:

- Make sure Toolbar > Split
  Link option is turned on.

![](GUID-23650AD2-38EC-461F-9C91-951DB824C9F1-low.png)

- Click on any part of the
  link aside from the start and end nodes.
- Depending on the node
  numbering setting (automatic vs. manual), you might be prompted to enter a new
  node number for the new node at the location of the split. If you are prompted,
  please enter a new node number.
- You can also right-click on
  any part of the link > Split Link > click on the desired location to
  split the link.

![](GUID-17B5EF9B-4EE3-4225-AD64-5C9B9C4A581E-low.png)
