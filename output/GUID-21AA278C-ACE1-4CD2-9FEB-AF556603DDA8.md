# TRIPENDS

Type = Logical

Default = F

This option tells the program whether or not to use trip end data in the
static estimation process. Use TRIPENDS=F if you do not
wish to incorporate trip end data, use
TRIPENDS=T if you do wish to use trip end data.
When using this option, the user must select either the
[AUTOTE](GUID-6043C302-4754-43B6-B9DB-A3832D5E59D4.html) or
[FILETE](GUID-32243635-3F01-4E72-9FB4-E02C0ECD6089.html)
option.
