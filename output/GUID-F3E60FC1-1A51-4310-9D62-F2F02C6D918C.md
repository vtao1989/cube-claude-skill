# Using Partial Trip Data in Analyst Drive

Including partial trip data in the dynamic estimation process is as easy
as including screenline or turncount data, and any combination of three types
of count data may be used. The first thing the user will need to do in order to
use partial trip data in the dynamic estimation procedure is produce a
corresponding partial trip data file for each user class in a similar manner to
screenline and turning count files.
