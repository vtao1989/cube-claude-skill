# Dynamic output matrix

When running a dynamic estimation, the user should select a
dynamic output matrix for each dynamic input matrix used in the problem. The
dynamic output matrix will contain the same format as the input matrix, and
will preserve warm up period matrix tabs when present (and when the
[WARMAT](GUID-535E2DC5-9C64-4A1A-835F-D475CA2E9C4C.html)option is selected).
