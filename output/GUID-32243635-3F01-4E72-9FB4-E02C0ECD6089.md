# FILETE

Type = Logical

Default = F

When using the
[TRIPENDS](GUID-21AA278C-ACE1-4CD2-9FEB-AF556603DDA8.html)
option, the user can provide static trip end data from a file using
[FILETE](GUID-32243635-3F01-4E72-9FB4-E02C0ECD6089.html#GUID-32243635-3F01-4E72-9FB4-E02C0ECD6089)=T
and linking an appropriate trip end file in the
Analyst Drive block diagram in Application
Manager.
