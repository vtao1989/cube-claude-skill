# Avenue dynamic turn count files

Analyst Drive can use dynamic turning counts in the
estimation process in conjunction with
Cube Avenue. The turn count files should be nearly
identical to the dynamic screenline count files but they must contain and extra
third column of nodes before the screenline count column since we require the
three nodes A, B, and C to determine the appropriate route-choice matrix.
