# Packaging external files

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-04826CC6-5BD5-4ADD-8854-9B00498AF5DF-low.mp4).

](GUID-04826CC6-5BD5-4ADD-8854-9B00498AF5DF-low.mp4)To generate a folder with all the files in the project that have
been externally linked:

Click on the Tools drop-down in the Application Editor
toolbar >  Pack
Externals  
![](GUID-CED7E268-35E2-4DB4-9C95-BD8FFE018276-low.png)

The Package Application will open:

1. Specify the max file size
   that will be allowed to copy
     
   ![](GUID-5E80AABC-80DD-4C7A-B531-1F5A03D65C53-low.png)  

   Note: Note: Use 0 if no
   size limit is wished to be applied.
2. Select/Unselect the
   "Also package externally linked images"
   as desired.
3. Click
   Package

The Package Results window will open summarizing any
external input, including image files (if selected)  
![](GUID-041B2C7D-0AC9-4192-9C79-8397BFE32DCF-low.png)

Click on Save to File to save this
summary as a text file in your device.

A Packages Folder will be added to your project directory
which will include all the externally linked files
  
![](GUID-3826A4EE-A0C8-4305-A637-DB37615F7D91-low.png)
