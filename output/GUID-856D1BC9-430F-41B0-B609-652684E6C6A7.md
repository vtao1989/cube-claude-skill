# Setting up Cluster Pools

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-118D93C3-51D6-4E7A-84A8-E6C5DE58A708-low.mp4).

](GUID-118D93C3-51D6-4E7A-84A8-E6C5DE58A708-low.mp4)Cluster pools correspond to a set of machines that have been set
up to take a number of processes and that can be easily accessed later when
running a model . To set up a Cluster Pool:

1. Click on
   Tools > Cluster Pool
   Management  
   ![](GUID-82FD8111-224D-41EE-B47D-F09D4133F9E7-low.png)
2. Click on Add
   Pool
3. Name the Pool by
   double-clicking on it
4. Click on
   Add Machine

   Note: The default TCP
   port 57364 1 process will be added to the server
5. Edit the number of
   processes to be handled by each machine by double-clicking on it.
6. Click on the Close symbol
   when all the machines for the Cluster pools have been set-up.

![](GUID-A1A03A6D-B0C3-4D94-AA88-CEE4EAA8E4BE-low.png)

When running a you will now be able to choose from the
created pools in the Batch Run composer.
  
![](GUID-EEBEFEC5-D242-4D6B-9841-5FC4FEAB964E-low.png)

To Remove a pool:

1. Open the Cluster Pool
   Management
2. Click on
   Remove Pool
