File Browser allows to easily navigate through your model's
project directory:

Navigating through a project:

File browser allows to visualize the folder structure of a
project through the tree structure and access the different folder contents.
You can navigate through the projects by:

- Clicking on the folder you
  want to navigate to in the tree structure.
- Double-clicking on the
  folder in the main section.
- Clicking on the Folder name
  in the breadcrumbs to jump in to that folder.
- Use the Back and Forward
  icons to go to the previous location opened.
- Use the Search box to Search
  for any files/folder inside the current folder

Changing the view:

- Click on the headers to sort
  the contents of the folder in descending/ascending order
- Clicking on the List view/
  grid view icon
  ![](GUID-B9B951F5-C50C-4624-9FE2-74CAECAF971D-low.png)will allow you to toggle between
  one or the other.

Filtering files

Clicking on the filter icon will open the available filter
categories and individual filters that can be applied:

![](GUID-D6D6AE0B-6B00-4446-90DD-99340B6AC424-low.png)

- Click on the box next to the
  file extension to apply any of the individual filters
- Click on the category name
  (e.g. CUBE 6) to automatically select all the filters under that category

To undo filters:

- Click on the selected filter
  or filter category
- Click on Reset

Adding files/folders

1. Click on the add icon to
   choose to add a new text file, folder or database(.SQLite) to the current
   folder.
2. Rename the file as desired.

Database file options

Selecting any database file type will open a Database tab
next to the Details section.

The datasets available in the database will be listed in a
tree structure. You can:

- Double-click or
  right-click > Open
  to open the dataset in the appropriate viewer/editor.
- Right-click > Removeto delete the selected dataset

  Tip: Use the
  Delete key to remove a dataset from the
  database.
- Right-click
   > Create > Create
  Network to add a network to the database
- Right-click on a network
   > Create > Transit
  Network/Non-transit network/ Junction Set/ Turn Penalty set/ Toll
  Matrix to add any of the mentioned datasets to the
  network
- Right-click on a network
   > Editto change the Maximum Zone Number Starting Node Number
  and other data of the network
