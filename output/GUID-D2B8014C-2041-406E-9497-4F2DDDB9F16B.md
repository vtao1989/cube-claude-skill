# Application Manager

Application Manager corresponds to the Application section
in the model tab in Project Explorer, which will list all the Applications and
Groups available in the currently opened project.

Use the
Filter... section to look for a specific application
or group listed in Application Manager  
![](GUID-52154A3F-4C93-4296-A42A-F5335A43DFC2-low.png)

Application Manager allows to visualize, organize, add,
import, remove and rename applications. Check the video tutorials for
additional information on
[creating or
opening applications](GUID-A4D9F034-6851-4CD5-BC70-D389647A65F6.html)
