# Print file & error codes

The print file will contain a brief summary of the program
run, along with a completion code (a
FILE\_ERROR value; see the error codes in the next
section) and completion time.

For non-MPI runs (i.e. the
[MPICORES](GUID-539DD601-CF95-495D-8993-D058970BF046.html) parameter is not set) then the print file will
contain information about each iteration of the program run including cost and
RMSE information. Print files from MPI-enabled runs will only contain summary
information for the estimation of each user class that is recorded by the
individual processes. All print files contain a completion summary providing
basic statics on how well the estimation performed, and also gives a completion
time in seconds.
