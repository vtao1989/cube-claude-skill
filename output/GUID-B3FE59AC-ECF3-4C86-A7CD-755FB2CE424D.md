# MPIHOST

Type = Character

Default = n/a

The
MPIHOST parameter defines a character string which
is used as the argument for the -host option of mpiexec. This will direct the
MPI process to run on the specified host machine which can be given by a
complete address or by the short name. Use this option to direct the
Analyst Drive program to use a specific node of a
cluster. This parameter has no default value.
