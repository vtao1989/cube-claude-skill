# Add/Update Non-Transit Modes

[

In this tutorial, we will look at how to add and update
non-transit modes.

1. Select the Non-Transit network dataset in the map table
of contents.

![](GUID-62A74441-1238-4CF3-AEB2-DFD893DF3665-low.png)

2. You can access the Transit System Data tool by:

- Right-clicking the
  Non-Transit dataset >
  Open Transit System Data.

![](GUID-12C08E06-E011-4F63-BF4E-F1F374BA849D-low.png)

4. To create a new mode > click on
Add Mode.

![](GUID-F4AB546C-1372-4100-A25F-C1F559B98DC5-low.png)

5. A new mode should appear at the bottom of the window
with a new node number. You can replace this value along with the Short Name
and Long Name of the new mode.

![](GUID-FDF07B65-5F12-418A-8D36-4DAC66611591-low.png)

6. Click
Save to confirm the changes.

Note: With the
Filter bar, you can filter out any mode by its
number, short name, or long name.

7. Select any existing line and use steps 5-6 to update its
changes.
