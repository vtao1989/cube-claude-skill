# Application Manager errors

If there was a simple error, it will be reported in the
print file of the
Analyst Drive program block with the
FILE\_ERROR message. In the case of an unknown error, the program may crash
quickly and not produce any output (or perhaps just a code that says the
program crashed for an unknown reason). In this case it is advisable for users
to run the program from the command line. For instructions, see Â[Running from the command line](GUID-FF93B941-C764-4503-905C-4C18914808F0.html#GUID-FF93B941-C764-4503-905C-4C18914808F0)Â.

Note: If after the
program has completed, the print file does not show the FILE\_ERROR message then
the program has crashed and was unable to produce it.
