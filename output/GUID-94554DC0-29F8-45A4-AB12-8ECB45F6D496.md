# DYNCONF

Type = Logical

Default = F

Use
DYNCONF=T if you wish to include count confidence
levels in the dynamic count files. When using
DYNCONF=T, a confidence value for each time
segment should be appended to each screenline/turncount row of the count file
after the final count value for that row. E.g., if there are 12 count values
per row, then there should also be 12 corresponding confidence values which
follow in order after the 12th count value in each row.
