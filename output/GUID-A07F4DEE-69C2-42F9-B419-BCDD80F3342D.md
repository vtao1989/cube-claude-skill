# What is Analyst Drive?

Analyst Drive is a matrix estimation program
available as a module in the
Cube software suite to licensed
Cube Analyst
users.
Analyst Drive uses state-of-the-art data assimilation
techniques to optimize trip origin-destination (OD) matrices for a given
traffic assignment and input traffic count data. While
Analyst Drive builds on the estimation tools provided
to the
Cube software suite by the
Analyst module, it is a completely new program with
advanced capabilities not previously available to the
Cube platform. A reference for comparing the
capabilities of
Analyst and
Analyst Drive is provided in Table 1. Some key
differences exist between some of the files that are used by the two programs,
for example
Analyst Drive uses a new ICP file format for the
Highway program which has been developed to dramatically reduce the file size
for large problems. A chart showing the file compatibility across the Cube
platform modules for
Analyst and the new
Analyst Drive is provided in Table 2.
