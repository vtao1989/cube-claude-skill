# Text Editor

Opening Text Editor

To open an empty text file go to the File Menu and select
New > Script/Text

To open an existing script go to
File > Open

Note: A script file filter (Matrix (\*.s, \*.py, \*.ctl, \*.bat) is available
in the Open window

To open an existing text file go to
File > Open

Note: A script file filter (Text (\*.txt, \*.log, \*.prn) is available in
the Open window
