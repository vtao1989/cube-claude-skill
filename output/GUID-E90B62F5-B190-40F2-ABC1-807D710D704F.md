# WARMUP

Type = Logical

Default = F

This option tells the program whether or not to use a warm
up period in the dynamic estimation. If using a warm up period in the
estimation, one should also be used in the
Cube Avenue assignment (in which case there should be
a negative start time in the packet log file).
