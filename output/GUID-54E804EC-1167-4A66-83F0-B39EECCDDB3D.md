CUBE lets you customize the location of the tabs/windows the
way you want it. This is useful when users work with multiple files or wish to
place on a different monitor.

1. Click and drag the tab you would like to position. This
should display a position anchor in the center of the current group of tabs,
which will allow you to position the new location of the window. If you would
like to move the current tab to an existing group of tabs, drag the tab to the
new group and the position anchor for that group will display.

![](GUID-AFBC5B97-64C9-438A-87E1-1D4647F1D65E-low.png)

2. To position a tab window relative to the main CUBE
window as a standalone tab, drop it on one of the outer positions in the
anchor.

![](GUID-02A51176-4FF7-4B43-BAA7-EADA4AD214E0-low.png)

![](GUID-EB6A9739-3A7D-43C8-8542-2180E5D4EFA2-low.png)

Tip: As you
are dragging the tab, CUBE will highlight the new position of the tab window.

3. To position a tab window relative to an existing tab or
group of tabs, drag the tab window to the target tab or group of tabs and the
position anchor for that group will display. Drop the tab into one of the outer
positions in the anchor.

![](GUID-0639D025-17F7-40C8-A31A-E97711241D78-low.png)

![](GUID-B652D78E-A6FA-447E-A54E-07F6327EA560-low.png)

4. To position the tab window outside of the existing CUBE
main window:

- When working with a single
  monitor, drag the tab bed window to the top of the main CUBE window.
- When working with multiple
  monitors, drag the tab window to the desired monitor.

![](GUID-D6A54AAF-3F75-446A-BFD1-72794D324480-low.png)
