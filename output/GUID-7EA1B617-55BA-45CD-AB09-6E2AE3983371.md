# Results

After performing dynamic estimation with partial trip
data, a section for reporting estimation results and statistics will be
included at the end of the report file. This data is very similar to that
provided for screenline counts and turning counts except that it provides both
the input and scaled observed counts, and that all statistics are reported by
matching against the scaled count value.
