# CubePy Virtual Environment in CUBE IDE

CubePy can be imported directly within the CUBE GUI. Python
scripts can be included in Application Editor through what called "CubePy"
programs. The default interpreter is the one coming with the CUBE installation,
but it is possible to define a different interpreter through the
Tools > Options > CubePy
window . At the current state of development, it is
not possible to set up a virtual environment as part of the project level
settings, therefore to use external packages with the default interpreter it is
necessary to install them manually through:
