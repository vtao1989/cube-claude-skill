](GUID-C40B71B5-650C-4500-8AEA-66E2F19FA28D-low.mp4)To display the shortest path from a node to another in a Map:

1. Right-click on Map
   Layers > Add Shortest Path
   Layer
2. Click on the
   Shortest Path Layer that has been added
3. Click on the Start Path
   symbol
   ![](GUID-D3B37791-3EAE-49EE-8EC7-71950C8C0395-low.png)in the GIS Editor toolbar
4. Go to the Map and select
   the origin node for the path.
5. Click on the symbolâs
   drop-down and select End Path.  
   ![](GUID-DB3267CF-7901-44E6-BBB6-61497664C31C-low.png)
6. Go to the Map and select
   the destination (End) node for the path.
7. You can adjust the
   Start Path and
   End Path
   nodes by selecting either of them in the toolbar and clicking
   on the new node.

To customize the shortest path visualization:

1. Go to the
   Shortest Path Layer Properties
   window > 
   Renderer
2. Right-click under Color to
   adjust the path color or color setting:

- Simple(default): will display one single color for
  the entire path  
  ![](GUID-3E1E4AE1-BC50-498D-A943-113356B1410A-low.png)
- Classbreaks: depending on the variable selected it will show several
  colors depending on the value distribution.
    
  ![](GUID-2ECB153B-7B28-4378-835A-59BC2B4480A3-low.png)
- Range:
  depending on the variable selected it will show a color gradient between the
  ones selected.
    
  ![](GUID-B6F1352B-F118-4EDD-AF7D-320CE36930E8-low.png)

Similarly you can adjust the width and width options:

- Simple(default):
  will display a singular width for the entire path
- Classbreaks: depending on the variable selected it will show
  different widths depending on the value distribution of the variable.
    
  ![](GUID-7B02B2CA-73CB-4BAF-9E14-197AE07000A3-low.png)
- Range:
  depending on the variable selected it will show a change in width between the
  ones selected in the range.
    
  ![](GUID-EBA7650B-EE91-4F19-A610-964E6C207FC5-low.png)
