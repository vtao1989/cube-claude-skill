# Edit Attributes for a Selected Feature (Link or Node)

In this tutorial, we will look at how to edit the attributes of both a
link and node.

[

**Your browser does not support HTML5 video.**

[Click here to play the video](GUID-F7606213-F463-4C7E-9547-D0144213B830-low.mp4).

](GUID-F7606213-F463-4C7E-9547-D0144213B830-low.mp4)

1. Select the highway link dataset in the map table of contents.

![](GUID-0B95F4AD-35B5-4BDF-A7A7-0CA85C44C056-low.png)

2. Confirm the Toolbar > Cursor tool is selected.

![](GUID-C13C8C32-AFB4-400C-BFB2-64F0AA97FDF8-low.png)

3. Select any link in the network and its attributes should show up in
the Feature Table. By default, the link should be highlighted in the table.

![](GUID-B7B818D3-DE0B-451B-9D31-B4E6557AADF1-low.png)

4. Double-click any attribute value to edit the attribute > enter a
new value.

![](GUID-D4F6D33C-666A-410A-8662-7BE55CE83193-low.png)

5. Click on the Commit Changes icon to confirm the edits.

![](GUID-D659CDD1-11A3-4ACC-A717-0BE876978C52-low.png)

Tip: You can revert changes any time before committing by
clicking on the drop-down arrow next to the Commit Changes tool > Revert
Changes.

![](GUID-71C25F54-AFFD-4535-AF13-4C52B943F139-low.png)

![](GUID-266A3768-7A2B-4B40-ADD3-FB4134E525F1-low.png)

6. Repeat steps #2-5 for the highway node dataset.
