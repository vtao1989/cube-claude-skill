# Overview

To perform static estimation in the simplest case of a
single OD matrix with default parameters, the user must at a minimum provide an
input OD matrix and
Highway-generated ICP and Screenline files. The
Highway-generated ICP file must be in the new format
(i.e. created using
FORMAT=1, see Figure 10). The only required
parameters in this case are
[ODTYPE](GUID-59278DA1-78AE-4A32-AE65-96AE07D46518.html)=0
and the
[WORKDIR](GUID-DD46DFCC-6C37-45AC-842C-964556F6ED8F.html)
parameter which should be set to a valid working directory folder path with
file-write permissions. For a slightly more complex estimation, some simple
options can be selected from the
Options menu that do not require additional input
such as using automatically generated trip ends by checking the boxes
TRIPENDS and
AUTOTE and using the screenline count confidences
present in the
Highway-generated screenline file by checking the
CNTCONF option box. The user may also alter the
default parameter values such as
[MATWGHT](GUID-03AB6DDA-D71E-4D9E-B6C3-0A7785770802.html) and
[DELTA](GUID-A647E346-83BB-4C5E-BBCC-7744B0031FB1.html), and
should consult Â[Parameters and Options](GUID-AE24CA39-878C-4F55-A6B4-2C6F9ACFB0F2.html)Â
on page 21 for information on how the various parameter values can affect the
optimization. The remainder of this section will deal with more advanced topics
in static estimation.
