# MATWGHT

Type= Real(+)

Default = 0.1

Range = 0.0-10,000

Prior matrix weighting factor. Should be small but greater than zero,
this parameter will affect how much the OD matrix undergoing estimation is
allowed to change. A value that is too large will cause the solution to go
unchanged, but a value that is too small could cause the solution to change too
much and allow unwanted solutions to arise.
