# Why Python

Python is one of the most popular general-purpose
programming languages, designed to be interpreted.

Its syntax is intuitive, and a very large number of
libraries are available, both open-source and from commercial software.

Python is strongly used worldwide in the field of Data
Science and Transport Modelling.
