The Compute tool in table editor lets you update field
values in a table, based on conditions. These conditions can be used to filter
rows to include in the update.

1. Click Toolbar
    > 
   Update  
   ![](GUID-319A9E27-850C-46B3-A471-FE4AEB01D10A-low.png)
2. Enter the expressions for
   updating the field values in the expression box on the top and the expressions
   for filtering the rows in the box below.
     
   ![](GUID-D1BF6747-F9E9-47A6-8965-93D8B7E7507F-low.png)  

   - Enter the name of the
     field whose value you want to change. Select the matching one from the
     autocomplete list.

     Tip:
     For a complete list of field suggestions use the shortcut:
     CTRL+SHIFT+C
   - Use the assignment
     operator â=â to assign the field its new value (numeric or non-numeric).
   - Enter a numeric or
     string value after the operator:
       
     ![](GUID-B00FE851-6F23-4725-BE47-43D63730D1E1-low.png)
3. Click
   Apply
   ![](GUID-D609C6A2-B3EB-42A1-A733-3138F77E62A4-low.png)

   - Whenever you apply a
     condition the status bar on the bottom right will show if the filtering process
     is still in progress.

     Tip: More than one field can be updated by separating the field
     expressions with a comma  
     ![](GUID-F1FB1221-0262-4C1E-93C4-D4C73EBF848B-low.png)
4. To clear the condition box,
   click on the drop-down next to Apply and select
   Clear
