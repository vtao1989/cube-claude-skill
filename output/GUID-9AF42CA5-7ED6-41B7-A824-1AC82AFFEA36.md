To add, remove and manage layers in a new or existing map in GIS Editor:

1. Go to the Map Layers
   section:
   1. Click on the Layers
      symbol or in the drop-down > Add
      Layerâ¦
   2. Right-click anywhere in
      Map Layers > Add
      Layerâ¦
2. The Add Layers window will
   open:
   1. Use the Browse
      symbol![](GUID-62D0327C-154B-4CFE-A2EB-902C46B8EEAC-low.png) to look for the database file you
      wish to load
   2. Click on the drop-down
      to navigate through the recently used databases.
        
      ![](GUID-EF040FAC-FD69-4F36-9632-ED97B1C388A0-low.png)
3. When a database has been
   selected the available datasets will be listed:
     
   ![](GUID-237F996A-A8F6-4711-B84A-DFBA79310A84-low.png)
4. Click on the datasets you
   wish to add to the Map
    > OK

Note: You can select multiple datasets using
CTRL+click

You can go back to the Add Layers window and add additional layer to the
Map.

- To hide/unhide layers:
  - Click on the checkmark
    to the left of the layer name to hide a layer
  - Click on the empty
    space to the left of the layer name to unhide a layer.
- Reloading a layer:
  - Right-click on the
    layer > Reload
  - Go to Add Layers and
    click on the Refresh symbol
    ![](GUID-E9E39BF8-9378-4237-8FE2-9642ECC86153-low.png)

- Duplicating a layer:
  - Right-click on the layer
    you wish to make a copy from > 
    Duplicate

Tip: Use the
CTRL+D shortcut to duplicate a layer.

- Removing a layer:
  - Right-click on the layer
    you wish to
    remove > Remove

  Tip: Use the
  Delete key as a shortcut.
- Renaming a layer:
  - Click on the layer and
    type in the new name
  - Right-click > Rename
    and type in the new name
