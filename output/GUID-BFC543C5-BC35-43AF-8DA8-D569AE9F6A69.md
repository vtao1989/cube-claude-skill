# TRNCNTS

Type = Integer(+)

Default = n/a

The number of turning counts for a dynamic estimation
problem. Only required for dynamic estimation, and only when using the
[GENERAL](GUID-537E9D14-8742-4648-BCD5-93CC253CD77F.html)=T
option. This parameter has no default value.
