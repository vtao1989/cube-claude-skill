Base maps from different providers are available in the GIS
Editor to customize your maps.

To add a base map:

1. Right-click on Map
Layers >  Add
Basemap

2. The base map Layer Properties window will open:

3. Select the map provider from the options available:

- Bing
- Mapbox
- Custom URL

  Note: Google URLs:

  - Google Roadmap:
  http://mt0.google.com/vt/lyrs=m&hl=en&x={x}&y={y}&z={z}

  - Google Terrain:
  http://mt0.google.com/vt/lyrs=p&hl=en&x={x}&y={y}&z={z}

  - Google Hybrid:
  http://mt0.google.com/vt/lyrs=y&hl=en&x={x}&y={y}&z={z}

4. Depending on the provider you will need to provide
additional information to retrieve the map.

For example for Bing maps you are able to choose from the
following options:
  
![](GUID-C85D6C57-D74B-4833-8C57-33B997A7E57F-low.png)

5. Go to the General tab to:

- Rename the Basemap
- Adjust Visibility and
  Transparency of the base map.

6. Click Apply
in the Data Source tab and Exit.
