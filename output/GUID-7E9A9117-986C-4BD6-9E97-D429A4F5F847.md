# STATS

Type = Logical

Default = T

The
STATS option allows the user to produce individual
count and summary statistics within the volume report file. Using this option,
the volume report will be augmented with additional statistics for each count
value and will also include advanced summary statistics for each user class.
The statistics provided by this option include various error, RMSE, GEH and
R2 measures.
