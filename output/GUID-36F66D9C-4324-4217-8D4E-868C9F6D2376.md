# ICP files

Analyst Drive requires new-style ICP
files generated by the
Highway program. The identifier
FORMAT defines the style of the ICP file to be
produced, and equals 1 for the new-style format or 0 for the old-style format
which is used by the original
Analyst program. The new-format ICP files have
multiple benefits in that they are typically much smaller in size than the old
format versions, and additionally the user can generate multiple new-format ICP
files in a single
Highway run as opposed to old-format ICP files which
require a seperate
Highway
run for each file produced. Figure 10 provides an example
script for generating multiple ICP files from a single run in the updated
Highway program available in
Cube
6.
