In this tutorial, we will look at how to add non-transit
legs.

![](GUID-DEBBE89D-F4EC-4937-BF6E-CBCE7921C3FB-low.png)

2. Click on the Toolbar > Create Non-Transit Leg tool.

![](GUID-3CDA9CCE-A589-4515-AEF5-D1EDE2B445C6-low.png)

3. To create a new non-transit leg:

- Click on an existing node,
  e.g. zonal node, to create the first non-transit stop.
- Continue clicking on the
  next stops in your new non-transit leg and CUBE will auto-route the line.
- Comparable to creating
  transit lines, you can create a direct link between non-transit stops by using
  CTRL + click.

![](GUID-9CCD0333-0487-4E95-AE11-0AC2CEDE2A6B-low.png)

- Right-click the last node or
  anywhere in the map window > Accept Sketch.

![](GUID-B164580D-3E6D-46A0-93F4-FA3200CC7544-low.png)

4. The non-transit leg name cannot be changed because it
incorporates the first and last stop nodes by design. You must specify the
non-transit mode.

![](GUID-25DFED98-B0C0-4DB2-904C-EEDA20C179BD-low.png)

5. Comparable to adding a new transit line, it is also
possible to copy the non-transit attributes from an existing leg by turning on
the copy attributes option and selecting a leg from the list. Click OK to
confirm the settings for the new non-transit leg.
