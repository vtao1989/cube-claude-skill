To generate a new application from an existing group:

1.Right-click on the
group > Save as
App  
![](GUID-B5C3FFA9-BD9D-4BAB-99CD-ACFD4F6BCE0D-low.png)

2.The Save Group as Application will open:

- Select the Application name
  and display name.
- The location in which the
  application will be saved will be shown in the Output directory section.
    
  ![](GUID-F9ADBEA4-402B-4FD0-BE93-CA03C1C92CD2-low.png)

Note: Display name corresponds to the name that will appear in the
Properties window and in the Breadcrumbs.

3. Click Save

The group will now appear in the application level in the Application
Manager tree.
  
![](GUID-89383407-AB5E-4442-A29F-FD59766EA54C-low.png)

A new Application with the given name will be available in the
Application folder in the Project file directory
  
![](GUID-A5A7BABB-78B4-4FD1-BB89-56C9AA59E9C3-low.png)
