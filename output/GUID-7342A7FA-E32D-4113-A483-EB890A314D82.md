# Creating a new CUBE Database with the CubeDatabase class

To create a new database create=True should be used, so that
the database will be created if it does not exist in that location. If the
database already exists, it will be re-created as a new empty database.

```
new_db = "output/Cubetown_new.SQLite"
db = cp.CubeDatabase(new_db, True)
```
