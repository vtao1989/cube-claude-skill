# MEMOPT

Type = Logical

Default = T

The
MEMOPT option instructs the program to perform
some routines using advanced algorithms which require a large amount of memory.
It can substantially speed up certain parts of the program, but requires a
minimum of
Zones2
\*TimeIntervals2\*
ScreenlineCounts
\*12 bytes of memory. If the option is set to true and not
enough memory is available to the program during execution an error code will
be thrown.
