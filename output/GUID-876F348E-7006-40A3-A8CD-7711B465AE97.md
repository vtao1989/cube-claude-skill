# DYNCLASS

Type = Integer(+)

Default = 1

The dynamic class number to estimate when the user desires
to estimate only a single user class from a multiclass problem. When running a
dynamic estimation using an
Avenue packet log file that was generated using
multiple user classes, the user may choose to only run the estimation for a
single user class from among those run in the assignment by using the
DYNCLASS parameter. To use this parameter, set
equal to the appropriate user class number corresponding to the
Avenue assignment that you wish to estimate, provide a
dynamic matrix and a dynamic count file corresponding to this class number, and
keep the
[NCLASSES](GUID-A93296A2-561D-4E9B-807E-3114ACB9B4D8.html)
parameter set to 1.
