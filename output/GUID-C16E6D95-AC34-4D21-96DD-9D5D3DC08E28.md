](GUID-D80E2619-F673-49E5-A4FE-998F002D49DC-low.mp4)It is possible to customize different aspects of how an
application looks:

- Modify color:
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > Colors
      
    ![](GUID-CF5CB7CE-A128-4E96-AF4A-8C7B5488628C-low.png)
  - From the Context menu:
    Right-click on an empty space in the Application Editor
    canvas > Colors
      
    ![](GUID-DD4A99FB-40F5-4F85-A68E-09C8C144929E-low.png)

1. Select the element whose
   color you wish to change >  Choose
2. The Default element colorâs
   window will open:  
   ![](GUID-13AD4FA7-F385-465F-B7BD-CCE17A03FDB7-low.png)  

   1. Default: How the element looks in
      the application
   2. Selected:
      How the element looks when you click on it
   3. Focused: How the element looks when
      a linked/related element is selected.
3. Select the color for
   Fill, Border, Shadow, Text
   and
   Text Shadow
4. Use the dragger to adjust
   the colorâs opacity
5. Click
   Update when done

- Modify font:
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > Font
  - From the Context menu:
    Right-click on an empty space in the Application Editor
    canvas > 
    Fonts  
  ![](GUID-0D8A87F0-4079-47EC-905C-45529D4D86BD-low.png)

1. Select the element whose
   font you wish to update >  Choose
2. The Select Font window will
   open:  
   ![](GUID-CF67938F-FDE5-4662-8E93-F93CBA16F8C3-low.png)
3. Select the
   Font, Font style, Size
   and
   Effects
   that you wish to apply
4. Click
   OK when done

- To save a group of color and
  font settings (Themes):
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > Themes > Save
  - From the Context menu:
    Right-click on an empty space in the Application Editor canvas
     > Themes > SaveName the theme and save it.

- To use another available
  theme:
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > Themes > Load
  - From the Context menu:
    Right-click on an empty space in the Application Editor canvas
     > Themes > LoadSelect the theme you want
   > Open

- To modify the color/ font of
  an individual element (program, file, annotation, etc.):
  - Right-click on the
    element > Color
     > Choose
  - Right-click on the
    element > Font
     > ChooseClick
  Update/OK when done
- To revert the element to use
  the same color and font settings as the rest of the application:
  - Right-click on the
    element > Color
     > Inherit
  - Right-click on the
    element > Font
     > Inherit
- Customizing file links (line
  rendering):
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > File links
  - From the Context menu:
    Right-click on an empty space in the Application Editor
    canvas > File links  
  ![](GUID-46CC454F-BFAA-490E-9F17-B8335F605630-low.png)

1. Select the
   Color, Thickness, Pattern and
   Curvature of the links.
2. Click
   Update

- Displaying grid:
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > Grid
     > Show Grid
  - From the Context menu:
    Right-click on an empty space in the Application Editor
    canvas > Grid
     > Show Grid

Note: When enabling the
grid to be displayed multiple options to customize the grid will be available
in the context menu like: Grid color, Grid size, Snap To, No Snap.

- Controlling which files are
  displayed next to the program:
  - Right-click on the
    program box >  Hide unused
    files to only display the file boxes that are being
    used.
  - Right-click on the
    program box >  Show all
    files to display all the potential files that can be
    linked to the program.
  - Right-click on the
    program box >  Show less
    files to only show the default program files.
  - Right-click >  Hide all
    files to only show the program box.
- Displaying specific file
  information:
  - From the toolbar: Click
    on the settings drop-down in the
    toolbar > File
    Info
  - From the Context menu:
    Right-click on an empty space in the Application Editor canvas
     > View
     > File..
  - Select if you want to
    display either:
    - File Description
    - File Names
    - File Sizes

You can find additional setting options in the Settings tool
and under View in the Context Menu.
