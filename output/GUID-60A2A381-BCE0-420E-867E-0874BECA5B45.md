# Using a warm start

If the
Avenue assignment has included a warm-up period, the
user should choose to use the warm start option for more accurate results in
early time periods. When using this option, Analyst Drive will generate warm-up
period route choice and OD matrices while processing the packet log file.
Analyst Drive uses these to generate a warm-up period
volume which is used to adjust the observed counts, as described in Chapter 2,
