# Schedule Multiple Scenarios for Model Run

[

For this exercise, we will look at how to perform multi-scenario run.
This process can be achieved through the Batch Run Composer and is fully
automated.

1. Open any project file which contains multiple scenarios.

![](GUID-2964A935-7372-4923-8754-FAD9E6EB8C4E-low.png)

2. Click on the Toolbar > Run to open the Batch Run Composer (BRC)
window.

![](GUID-F6C8B858-6EAF-4C03-A441-C9DE1AAF5B85-low.png)

![](GUID-F152B60C-EA0D-4BA0-BE5D-6EDFE283DC58-low.png)

- You can also access the BRC
  from the Project Explorer.

![](GUID-007721A2-E019-4651-8085-70195AF51D54-low.png)

- Tip: If you have the Scenario Manager open for a particular
  scenario, the BRC can also be access this way.

![](GUID-602CF40E-ABC2-4FF8-8D2A-6312343A0B61-low.png)

3. With the Bath Run Composer open, select any application.

![](GUID-09D71164-93E1-4103-9568-E94DDB4D8E0D-low.png)

4. Now select all relevant scenarios to schedule the multi-scenario run.

![](GUID-59C8A48B-F30C-4E09-884B-04765B0705CF-low.png)

5. Click Run. In this exercise, we will not cover the other two options.

![](GUID-94978514-DBC8-49B2-88EB-BA2620AB3D1D-low.png)

6. When completed the Run Status Monitor window should provide
information on the scenario name and code, start/end time for each scenario,
its status, and messages for warning/errors. You can click on the latter to see
the results for each scenario and any warning or errors that may have appeared.
Click Close.

![](GUID-6CD5CEDA-18D3-4760-BEAE-37D3F8F92100-low.png)
